# Matrices

Как решить ниже задачи?
2500. Delete Greatest Value in Each Row 
1572. Matrix Diagonal Sum
1260. Shift 2D Grid 
867. Transpose Matrix 
766. Toeplitz Matrix 

Есть “трюки и паттерны” для таких задач с матрицами, то есть типовые подходы, которые помогают решать множество задач одной техникой, вместо того чтобы придумывать всё с нуля.

## Трюки

1. Итерация по матрице. 
Стандартный способ: двойной цикл по строкам и столбцам.

```cpp
for(int i=0;i<n;i++){
    for(int j=0;j<m;j++){
        // работа с matrix[i][j]
    }
}
```

2. Обход по диагонали

Для задач вроде Matrix Diagonal Sum.
Основной трюк: элементы главной диагонали имеют индексы matrix[i][i].
Элементы побочной диагонали: matrix[i][n-1-i].

3. Сдвиги/повороты

Для Shift 2D Grid или Transpose Matrix.
1) Transpose — просто меняем matrix[i][j] на matrix[j][i].
2) Shift — можно перевести 2D индекс в 1D, сдвинуть, потом обратно.

4. Максимум/минимум по строке или столбцу

Например, Delete Greatest Value in Each Row.
Трюк: пробегаем каждую строку, находим максимум и суммируем/удаляем.

5. Паттерны “проверка условия по всем соседям”

Например, Toeplitz Matrix:
Все элементы диагонали должны быть одинаковыми.
Трюк: matrix[i][j] == matrix[i-1][j-1] для всех i>0, j>0.

Если освоить эти “шаблоны”, задачи с матрицами перестают быть страшными, даже если числа разные.

## Дополнительно полезные трюки для матриц:

- Двойной цикл: почти все Easy задачи можно начать с for i, for j.
- Преобразование 2D ↔ 1D: удобно для сдвигов, вращений, Flatten.
- Проверка соседей: для паттернов, последовательностей, равенства.
- Транспонирование / разворот: часто используется в поворотах или смене осей.
- Суммы / максимум / минимум по строкам и столбцам: частый приём в “сборе данных” из матрицы.
