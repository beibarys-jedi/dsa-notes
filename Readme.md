# Readme


This repo for dsa notes.


# Create new program

create new `C++` program with following content

```c++
#include <iostream>
using namespace std;

int main() {
    cout << "hello" << endl;
    return 0;
}
```

# Run project
below simple command lines with compilation

```shell
clang++ main.cpp -o main
./main
hello
```
